### Desktop (>1024px)
- Side-by-side map and predictions
- Full navigation with user info
- Large hero section with floating cards

### Tablet (768px - 1024px)
- Stacked map and predictions
- Simplified navigation
- Adjusted grid layouts

### Mobile (<768px)
- Single column layout
- Compact navigation (logout only)
- Simplified hero section
- Full-width cards
